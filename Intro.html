HTML Basics

HTML Documents

  • All HTML documents must start with a document type declaration: <!DOCTYPE html>
  • Every html douments begins with <html> and ends with </html>. This is the root element of every HTML document. Everything inside this tag is considered part of the HTML document.
  • There is another tag <head> which contains meta-information about the HTML document that is not visible to the user. It typically includes things like:
    • The document's title (<title>)
    • Meta tags like charset, viewport, etc.
    • Links to external files like CSS stylesheets or JavaScript files.
  • <body> contains the actula content that will be displayed on the webpage (headings, paragraphs, images, links, etc.)

Key points about <!DOCTYPE>

  • Location: It should be placed at the very beginning of the document, before the <html> tag.

  • Not case sensitive: The declaration can be written in any case, but it is commonly written in uppercase as you have already seen.

  • HTML5 Simplicity: Unlike earlier version of HTML (like HTML 4.01 or XHTML), which required longer declarations pointing to DTDs (Document Type Definitons - are a set of rules or pecifications that define the structure and allowed elements and attributes for an HTML or XML document.), the HTML <!DOCTYPE> is much simpler.

      For example HTML 4.01 DTD was like the following:

    • DTD for HTML 4.01 Strict: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
    • DTD for HTML 4.01 Transitional: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

  • Purpose: It triggers the browser to render the page in standards-compliant mode (Standards-compliant mode is the rendering mode where the browser follows the official web standards as defined by the World Wide Web Consortium (W3C) for HTML, CSS, and other web technologies.), rather than "quirks mode (Quirks mode is a rendering mode used by browsers for backward compatibility with older web pages that were created before modern web standards were established.)" (which emulates older browsers' behavior).

HTML Elements

The HTML element is a fundamental building block of an HTML document. It is a combination of an opening tag, content, and a closing tag (optional for some elements).

An HTML element consists of following parts:

  1. Opening Tag: Marks the beginning of the element. It is enclosed in angle brackets, e.g., <tag-name>.
  2. Content: The actual content enclosed by the element.
  3. Closing Tag: Marks the end of the element. It is similar to the opening tag but includes a forward slash (/), e.g., </tag-name>

<tag-name>Content goes here<;tag-name>

Types of HTML Elements

  1. Block-level Elements: these elements create a block on the page, often causing content to appear on a new line. They typically define larger sections of a page, like paragraphs or divider. Example: <div>, <p>, <h1>, <ul>.

  2. Inline Elements: These elements do not start on a new line; they flow within the content. They usually define smaller sections, like text or links. Examples: <span>, <strong>, <em>.

  3. Self-closing Elements: These elements don't have a closing tag. Instead, they are self-contained in the opening tag. Examples: <img>, <br>, <hr>, <input>, <meta>

Common HTML Elements

  1. <html>: Represents the entire HTML document.

    2. 
                        <html>
                            <!-- Content goes here -->
                        </html>

  2. <head>: Contains meta-information about the document, such as title, character set, and links to stylesheets ro scripts.

    3. 
                        <head>
                            <meta charset="UTF-8">
                            <title>Page Title</title>
                        </head>

  3. <title>: Specifies the title of the web page, whcih appears in the browser's title bar or tab.

    4. 
                        <title>My Webpage</title>

  4. <body>: Contains the content of the web page, such as text, images, links, etc. 

    5. 
                        <body>
                            <h1>Welcome to My Website</h1>
                            <p>This is a paragraph of text.</p>
                        </body>

  5. <h1> to <h6>: Define headings of different levels/size. <h1> is the highest (most important) level, and <h6> is the lowest.

    6. 
                        <h1>This is a heading level 1</h1>
                        <h2>This is a heading level 2</h2>

  6. <p>: Defines a paragraph.

    7. 
                        <p>This is a paragraph of text.<p>

  7. <a>: Defines a hyperlink.

    8. 
                            <a href="https://www.example.com">Click here</a>

  8. <ul> and <ol>: Define unordered (bulleted) and ordered (numbered) lists, repectively.

    9. 
                            <ul>
                                <li>Item 1</>
                                <li>Item 2</>
                            </ul>

                            <ol>
                                <li>First Item</>
                                <li>Second Item</>
                            </ol>

  9. <img>: Embeds an image in the document. It's self-closing and requires an src attribute for the image source.

    10. 
                            <img src="image.jpg" alt="Description of image">

  10. <div>: A generic container element, often used for grouping or styling content. 

    11. 
                            <div>
                                <p>This is inside a div element.</p>
                            </div>

  11. <span>: An inline container used to style a part of the content without affecting its layout.

    12. 
                            <span style="color: red;">This is red text.</span>

  12. <form>: Defines an HTML form for collecting user input.

    13. 
                            <form action="/submit" method="POST">
                                <input type="text" name="username">
                                <input type="submit" value="Submit">
                            </form>

  13. <table>: Defines a table. Inside it, your can have <tr> (table rows), <td> (table data cells), and <th> (table header cells). 

    14. 
                            <table>
                                <tr>
                                    <th>Header 1</th>
                                    <th>Header 2</th>
                                </tr>
                                <tr>
                                    <td>Data 1</td>
                                    <td>Data 2<td>
                                </tr>
                            </table>
